# Psychological Portfolio
### Haki Nafsi by Nour

*A personal website for Nour Abou Zaki, a Licensed Clinical Psychologist, built with vanilla HTML, CSS, and JavaScript.*

#### Pages:

1. Home (home.html): Hero section, professional overview, and contact form

2. About (about.html): Background, qualifications, and clinical experience

3. Services (services.html): Therapeutic approaches, professional roles, and assessment tools


#### Features:

- Responsive design with mobile hamburger menu

- Contact form that sends messages directly via WhatsApp

- Smooth scroll navigation

- Active link highlighting per page
